# Scala Project

```bash
# generate Scala maven module project
mvn org.apache.maven.plugins:maven-archetype-plugin:RELEASE:generate \
  -DarchetypeGroupId=net.alchim31.maven -DarchetypeArtifactId=scala-archetype-simple -DarchetypeVersion=1.7
  -DinteractiveMode=false -DgroupId=org.matihost.learning -DartifactId=scala-samples -Dversion=0.0.1-SNAPSHOT
```
